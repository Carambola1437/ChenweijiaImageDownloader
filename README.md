## 介绍
这是一个恶搞同学的项目，为何要把它发布在github呢？因为这样可以让他**更开心**😁。

到0.3版本，都是使用python的easygui编写，编辑器为Pycharm，连tk都懒得用。现在使用VsCode所以代码肯没有那么整齐。

这次的0.4版本将直接用控制台，虽然有些丑陋，但是启动速度快，代码好写。
